Works with public share links only (in Gemini: Share & export → Share → Create link). The page is fetched and rendered through the free Jina Reader service, so very heavy use may hit its rate limit — if that happens, wait a minute and retry.

Google “AI Mode” chats (share.google/aimode/…)

Google blocks tools from reading AI Mode pages remotely, so those links can't be pasted above. Instead you run a tiny snippet on the page itself. Pick the setup for your device — you only do it once.

📱 iPhone / iPad — Shortcut from the Share menu
  1. Open the Shortcuts app → ＋ → Add Action.
  2. Search for “Run JavaScript on Web Page” and add it. Tap the script box, delete the sample, and paste the code you copy here:
  3. Below it add two more actions: “Save File” (turn Ask Where to Save on so you can pick Files, or off to dump into iCloud/Downloads). The file's contents are the JavaScript result; set the extension to .html if asked.
  4. Name the shortcut e.g. Save AI chat, and make sure “Show in Share Sheet” is ON (shortcut settings), with Safari web pages allowed as input.
  5. To use it: open the chat in Safari, let it load, tap Share → Save AI chat. Done.

If it saves a file containing only NO_CONVERSATION_FOUND, the page hadn't finished loading — scroll to the bottom of the chat and run it again.

🤖 Android — saved-bookmark trick
  1. Copy the snippet:
  2. In Chrome, bookmark any page (tap ☆). Then open ⋮ → Bookmarks, tap-and-hold that bookmark → Edit.
  3. Set Name to saveai and paste the copied snippet into the URL field. Save.
  4. To use it: open the AI Mode chat, let it load, tap the address bar, type saveai, and tap the bookmark suggestion that appears. It runs on the current page and downloads the HTML.

Firefox for Android works the same way. (Chrome strips javascript: when you type it directly, which is why it has to live in a saved bookmark.)

🆘 Any phone, zero setup — Print → PDF

Not HTML, but instant: open the chat, choose Share → Print, then pinch-zoom the preview and tap the share/save icon to Save as PDF. Captures the whole conversation for archiving.
