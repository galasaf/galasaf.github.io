A collection of small projects and tools.

AI Coding Assistant Field Manual 📗

A complete guide to Claude Code and GitHub Copilot — from first session to expert configuration, workflow design, and fleet engineering. 26 parts, glossaries, troubleshooting, and printable cheat sheets.

Almost Thesaurus

An AI-powered thesaurus that finds uncommon, niche, or exotic words for a query instead of the first bland synonym.

Language Coach 🗣️

A voice-only English tutor — talk out loud with an animated avatar that teaches the most common words first, skips the ones you've already shown you know, and circles back to words you haven't used in a while.

Room Fitter

Plan a room's layout and check what furniture actually fits — draw the floor plan, place objects, auto-arrange by priority.

Mortality Pullforward Explorer

Models how COVID's mortality pullforward effect changed survivors' life expectancy — assume a pullforward curve, or calibrate one from assumed excess deaths.

Domain Genie 🧞

Describe an idea and get clever, punny domain names — each instantly checked live to see which ones are actually free to register.

Gemini Chat → HTML

Turn a Gemini share link (or Google AI Mode chat) into a standalone HTML file you can save and read offline, on any device.

Slouch Detector

A webcam-based deep learning model that predicts slouching vs. good posture in real time.

BreathView 👶

Point a camera at your sleeping baby and see the invisible — motion magnification makes each breath and heartbeat clearly visible, with live breaths/min & BPM. Not a medical device.
